Data Generator

With this tool Test or Demo data can be generated into the neo4j database. You can see it as a Swiss Army knife to build a fake data set.
The tool is inspired by GraphGen and the faker libraries (java faker). To make it possible to generate larger datasets the person name generator (firstname, lastname and fullname) is implemented in a different way.
